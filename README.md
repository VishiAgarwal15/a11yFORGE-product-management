# a11yFORGE Engine – Product Management

This repository documents product ownership work for the a11yFORGE Engine,
an end-to-end accessibility automation platform.

## What is a11yFORGE?
a11yFORGE is designed to help teams detect, prioritize, remediate,
and verify accessibility issues at scale.

## Core Components
- **WAS** – Accessibility detection and rule-based scanning
- **GAIA** – Decision intelligence and issue prioritization
- **KASHA** – Workflow execution, ownership, and verification

## My Role
- Defined MVP scope and in-scope vs out-of-scope decisions
- Translated scope into epics and user stories
- Managed dependencies and execution risks
- Maintained an execution-ready backlog for delivery
